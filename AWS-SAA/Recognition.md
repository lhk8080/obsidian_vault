> AWS 완전 관리형 AI 서비스

- 이미지와 동영상 분석 > 객체, 사람, 얼굴 텍스트 등을 인식
- ML 모델을 직접 개발하지 않고 이미지/영상 분석 가능
- Use Cases : 
	- 데이터 라벨링
	- 텍스트 탐지
	- 얼굴 탐지 및 분석(감정, 성별, 나이 등)
	- Content Moderation (부적절한 콘텐츠 감지)

---
### Content Moderation
> 부적절하거나 공격적인 컨텐츠를 AI가 자동으로 탐지

- 방송, 소셜미디어, 광고 등에서 안전한 사용자 경험을 제공
- `Minimum Confidence Threshold` : 최소 신뢰도를 설정 > 특정 값 이상만 부적절한 콘텐츠로 판단
- `Amazon A2I` : 확실하지 않은 경우, 사람이 최종 판단 할 수 있도록 하는 시스템 (human in the loop)
